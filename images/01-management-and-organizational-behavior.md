# Figures and Visual Elements — Chapter 01: Management and Organizational Behavior

## Figure 1: The Hawthorne Works Experiment Timeline

**Location:** Section 1 opening (approximately 500 words into chapter).

**Type:** Timeline/process diagram.

**Content:**
```
Week 1: Baseline measurement (lights at normal level)
        → Output: X relays per worker per hour
        
Week 2-4: Increase light intensity by 50%
        → Output: X + 8% relays per worker per hour (↑)
        
Week 5-7: Decrease light intensity to 20% below baseline
        → Output: X + 12% relays per worker per hour (↑)
        
Week 8-10: Return light to baseline level
        → Output: X + 15% relays per worker per hour (↑)
        
Week 11+: Stop observation/note-taking
        → Output: X - 5% relays per worker per hour (↓)
```

**Narrative purpose:** Visually conveys the counterintuitive finding — each physical manipulation, including the return to baseline, produced increased output until observation stopped.

**Caption:** "The Hawthorne effect: productivity increased each time light changed, including when it decreased. The variable was not the light. The variable was attention."

**Content:**

```
         value to others)
       ↑

  ↑

```

**Annotation:** Arrow pointing to the diagram: "These functions are not a strict hierarchy. People prioritize them differently. Remove any one and consequences appear."

**Caption:** "Work serves five functions. All matter. None is dispensable."

## Figure 3: Three Levels of Management and Skill Emphasis

**Location:** Section 3, after definition of management.

**Type:** Three-column comparison table or pyramid.

**Content (Table format):**

| Level | Primary Activities | Time on Planning | Time on Directing | Time on Controlling | Technical Skills | Human Skills | Conceptual Skills |
|-------|-------------------|------------------|-------------------|---------------------|------------------|--------------|-------------------|

## Figure 4: Schools of Management Thought — Timeline and Key Shift

**Location:** Section 4 (Schools of Thought).

**Type:** Timeline with annotations.

**Content:**
```
1890s-1920s: SCIENTIFIC MANAGEMENT
(Taylor, Gilbreth)
Focus: Efficiency, procedure, measurement
Problem: Ignored human dimension
└─→ Assumption: People respond only to pay/coercion

        ↓ Hawthorne Studies (1924-1932) reveal observation 
          changes behavior ↓

1930s-1950s: HUMAN RELATIONS
(Mayo, Roethlisberger)
Focus: Morale, social ties, satisfaction
Problem: Assumed aligned interests; conflict ignored
└─→ Assumption: Happy workers = productive workers

        ↓ Realized organizations are complex systems ↓

1960s-1970s: SYSTEMS THINKING
(Nadler, Tushman, Katz)
Focus: Interconnections, feedback loops
Problem: Too abstract; hard to apply
└─→ Assumption: Change one part, multiple parts move

        ↓ Realized effectiveness depends on context ↓

1970s-1990s: CONTINGENCY THEORY
(Burns & Stalker, Fiedler)
Focus: Fit between organization and environment
Problem: Says "it depends" but doesn't say what
└─→ Assumption: No universal best way

        ↓ Digital era demands evidence, not assumption ↓

2000s-PRESENT: EVIDENCE-BASED MANAGEMENT
Focus: What research actually shows; revision as evidence changes
Problem: Organizations don't always use evidence
└─→ Assumption: Test, measure, adjust
```

**Narrative purpose:** Shows progression from single-variable thinking (efficiency alone) to multivariable thinking (systems, context, evidence). Explains why modern management textbooks look different.

**Caption:** "Management thinking has shifted from efficiency-only to evidence-grounded. Each shift added complexity and accuracy."

## Figure 7: The Hawthorne Effect — Original Claim vs. Reanalysis

**Location:** Section 4, in the "Human Relations" school discussion.

**Type:** Side-by-side comparison; could be two bar charts or a narrative diagram.

**Content:**

**Original Claim (Mayo, 1933):**
```
Productivity Increase

Observation Effect: ████████████ (85% of increase attributed to being watched)
Other Factors:     ██ (15%)
```

**Reanalysis (Levitt & List, 2009; Fernandez-Cornejo et al., 2016):**
```
Productivity Increase

Wage Incentives:     ██████████ (40-45%)
Selection Effects:   ████████ (25-30%)
Supervision Changes: ██████ (15-20%)
Observation Effect:  ████ (5-10%)
```

**Narrative purpose:** Correction by visualization. Shows that the effect is real but smaller, and other variables were more powerful. Teaches the reader: "Original claims get revised. Stay skeptical."

**Caption:** "The Hawthorne effect is real, but the original study overstated it. Later analysis revealed that wage changes, which workers in the study received, explained more of the productivity increase than observation alone."

## Figure 8: Work-Disengagement Diagnostic Framework

**Location:** End of Section 5, as a practical tool.

**Type:** Decision tree or diagnostic flowchart.

**Content:**

```
PROBLEM: An employee is disengaged (low output, poor quality, withdrawn).

├─ BLOCK 1: Individual Psychology
│  ├─ Are they capable? (Test, interview, observe work)
│  ├─ Do they understand success? (Check role clarity, feedback)
│  └─ → If issue here: coach, clarify expectations, or move to different role
│
├─ BLOCK 2: Task Design
│  ├─ Is the task too easy? (Bored, unchallenged)
│  ├─ Is the task too hard? (Frustrated, helpless)
│  ├─ Does it use their strengths? (Misalignment)
│  └─ → If issue here: redesign work, add variety/challenge, or reassign
│
├─ BLOCK 3: Organization Design
│  ├─ Does the structure create obstacles? (Bottlenecks, ambiguity)
│  ├─ Are decision authorities clear? (Confusion, frustration)
│  └─ → If issue here: restructure, clarify reporting, add resources
│
├─ BLOCK 4: Organizational Processes
│  ├─ Does the person get feedback? (Visibility, recognition)
│  ├─ Is there psychological safety? (Can they speak up?)
│  ├─ Does leadership model engagement? (Hypervisibility, trust)
│  └─ → If issue here: improve communication, model engagement, build safety
│
└─ BLOCK 5: Management
   ├─ Does the manager understand the root cause?
   ├─ Is the manager present and available?
   └─ → If issue here: manager development, one-on-one time, or management change
```

**Narrative purpose:** Makes the five-block model actionable. Shows a manager how to diagnose (move through blocks systematically) and where to intervene (different solutions for different blocks).

**Caption:** "Disengagement usually has multiple causes. Moving through the five blocks helps you find where to intervene."
